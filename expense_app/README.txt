אפליקציית מעקב הוצאות (פרטית עם סיסמה)

מה זה עושה:
- התחברות עם סיסמה (רק מי שיודע אותה נכנס)
- העלאת חשבוניות PDF/תמונה
- זיהוי תאריך וסכום כשאפשר, ותמיד אפשר לתקן ידנית
- דשבורד סיכום הוצאות לפי חודש + פירוט
- ייצוא CSV

הפעלה מקומית:
1) התקיני Python 3.10+
2) בתיקייה, הריצי:
   pip install -r requirements.txt
3) בקובץ:
   .streamlit/secrets.toml
   החליפי את APP_PASSWORD לסיסמה חזקה
4) הריצי:
   streamlit run app.py
